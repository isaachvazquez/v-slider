About v slider

V SLIDER is a jQuery slideshow plugin.

Get it free on github.

How it works

Follow these 3 steps

1.) Include the vslider_styles.css file in the head of the document.

<head>
 <link rel="stylesheet" href="path/to/vslider.css">
</head>

2.) Simply add the HTML to the page.

<div class="vslider my_slider">
  <div class="vslider-items">
    <img src="image.png">
    <img src="image.png">
    <img src="image.png">
    <img src="image.png">
  </div>
  <button class="vslider-button js-vslider-previous">
  <button class="vslider-button js-vslider-next">
</div>

3.) Include the vslider.js file before the closing </body> tag.

 <script src="path/to/vslider.js"></script>
</body>

4.) Call the javascript below, and voala!


var el = vs_getAll('.slider1').vs_first();
var vs1 = new VSlider(el, {});
