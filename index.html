About v slider

V SLIDER is a smooth, sleek, and simple slideshow plugin for jQuery built around ease-of-use. It supports a declarative initialization style that allows full customization without any scripting. Simply include the plugin, declare your markup, and V SLIDER does the rest.

crisp. clean. comfortable.

so why should you use v slider?

Because you're cool.
And V SLIDER is cool, so it only makes sense.

Features

  • Customizable
  • Responsive
  • Linkable. Either button or full-width links.
  • Pretty.
  • Legit.

How it works

Simply add the HTML to the page. Call the javascript below, and voala!

(function ($) {
	$('.v_slider').v_slider();
})(jQuery);
