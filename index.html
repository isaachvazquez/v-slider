V SLIDER

About v slider

V SLIDER is a jQuery slideshow plugin.

Get it free on github.

How it works

Follow these 3 steps

1.) Include the v_slider_styles.css file in the head of the document.

<head>
 <link rel="stylesheet" href="path/to/v_slider_styles.css">
</head>

2.) Simply add the HTML to the page.

<div class="v_slider my_slider">
 <img src="image.png">
 <img src="image.png">
 <img src="image.png">
 <img src="image.png">
 <div class="v_slider_controls">
  <a href="#" data-action="previous" class="v_slider_back_button">
   <img src="v-slider-images/v-slider-left-arrow.png">
  </a>
  <a href="#" data-action="next" class="v_slider_next_button">
   <img src="v-slider-images/v-slider-right-arrow.png">
  </a>
 </div>
</div>

3.) Include the v_slider.js file before the closing </body> tag.

 <script src="path/to/v_slider.js"></script>
</body>

4.) Call the javascript below, and voala!

(function ($) {
 $('.my_slider').v_slider();
})(jQuery);
